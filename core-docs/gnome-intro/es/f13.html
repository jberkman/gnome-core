Querido entusiaste de GNOME,

Este documento acompaña a la versión alfa de GNOME, que salió en mayo de 1998. En este momento GNOME es un conjunto de programas interesante y útil, pero está bajo desarrollo muy activo. Esto significa que encontrará muchos programas que están incompletos y algunos errores que no son de esperar en un producto maduro.

Habiendo dicho esto, GNOME ya es un producto excitante, y muchos de sus programas son bastante útiles.

Qué es GNOME

NOTA: Si se cansa de las explicaciones y quiere avanzar con GNOME de forma práctica, por favor omita el resto de este prefacio y lea el Capítulo 1 inmediatamente.

GNOME es un proyecto que pretende proveer un entorno de usuario completo, consistente, avanzado y libre para los sistemas UNIX. Esto incluye:

  • Una base de aplicaciones — esto consiste en una guía de estilo para las aplicaciones junto con un juego de bibliotecas que obliguen a cumplir los requisitos de estilo. Por ejemplo, las bibliotecas GNOME le permiten al programador invocar menúes estándar y cuadros de diálogo.

  • Un administrador de archivos — el administrador de archivos GNOME está basado en el bien conocido GNU midnight commander, que ha sido enriquecido con una interface gráfica que cumple con GNOME–.

  • Un panel — este reemplaza a las barras de botones– provistas a veces por los administradores de ventanas de UNIX. El panel también permite ejecutar programas en pequeños cuadrados en el panel, tanto tragándose programas X Window como ejecutando applets escritos especialmente.

  • Soporte para administración de la sesión — se notifica a las aplicaciones para que puedan guardar información de estado cuando un usuario termina su sesión: la próxima vez que se inicie una sesión, estas aplicaciones seguirán desde donde las había dejado.

  • Un conjunto de aplicaciones GNOME — estas han sido escritas (o adaptadas) para usar las características GNOME, y por lo tanto presentan una apariencia y un comportamiento consistente.

Hay muchas más facetas de GNOME, y se puede encontrar mucha información en la página web de GNOME.

GNOME es un proyecto GNU, comenzó en 1997, y se ha desarrollado muy rápidamente. GNOME se puede pensar como un acrónimo que significa GNU Network Object Model Environment.(Entorno de Modelo de Objetos de Red GNU)
