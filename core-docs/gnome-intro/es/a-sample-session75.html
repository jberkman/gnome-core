Usando el soporte de internacionalización de GNOME

GNOME tuvo soporte de internacionalización (llamada a veces i18n) desde las fases de desarrollo más tempranas. Para hacer que un programa GNOME le de los mensajes en su idioma favorito, simplemente configure su variable de entorno LANGUAGE con el código de dos letras que pertenecen a ese idioma.

Por ejemplo, el comando gnome-hello-2-i18n invoca la variante gnome-hello que incluye soporte para internacionalización. Si quiere que este programa (o cualquier aplicación GNOME) muestre sus menúes y otra información en Español, puede ingresar (para el shell Bourne o bash):

$ LANGUAGE=es

$ export LANGUAGE

$ gnomecal &
