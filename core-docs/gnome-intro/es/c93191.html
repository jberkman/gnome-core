Apliaciones de propósito general

Editor hexa GNOME — GHex

El editor hexa GNOME es un editor para archivos no-ASCII. Se puede invocar desde el panel o ingresando

$ ghex &

Figura 2-9. Editor hexa GNOME: Barra de menú inicial

Figura 2-10. Editor hexa GNOME: archivo cargado

Usar ghex es bastante directo, y tiene un manual disponible en el navegador de ayuda GNOME.

gEdit

gEdit es un editor de texto simple. Se puede invocar desde el panel o ingresando

$ gedit &

Figura 2-11. gEdit: dos archivos abiertos simultáneamente

Terminal UNIX de GNOME (gnome-terminal)

La terminal UNIX de GNOME es un remplazo compatible con GNOME de xterm. Se puede invocar desde el panel o ingresando

$ gnome-terminal &
