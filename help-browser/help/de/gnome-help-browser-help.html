Dokumentation zum GNOME-Hilfebrowser

Der GNOME-Hilfebrowser ist eine Schnittstelle zu verschiedenen Arten von Information auf Ihrem Computer und auf anderen Computern, die durch das Internet erreichbar sind. Dieses kurze Dokument beschreibt die Hauptfunktionen des GNOME-Hilfebrowsers.

Kommandozeilen-Optionen 

      -d, --debug                Diagnoselevel, höhere Werte führen zu
                             mehr Ausgabe
  -h HÖHE                    Höhe des Fensters
  -w BREITE                  Breite des Fensters
  -x X                       X-Position des Fensters
  -y Y                       Y-Position des Fensters
      --display=DISPLAY      Zu verwendendes X-Display
      --no-xshm              X-Erweiterung für gemeinsam nutzbaren
                             Speicher nicht verwenden
  -?, --help                 Diese Hilfeliste ausgeben
      --usage                Eine kurze Hilfemeldung ausgeben
  -V, --version              Programmversion ausgeben

Menüs

Der GNOME-Hilfebrowser hat mehrere Menüeinträge am Kopf des Anwendungsfensters. Ihr Verhalten sollte selbsterklärend sein.

Werkzeugleiste

Die Werkzeugleiste hat mehrere Knöpfe, um die Verwendung des Hilfebrowsers komfortabler zu machen. Die Knöpfe Vor und Zurück lassen den Browser die Chronik von in dieser Sitzung bereits betrachteten Seiten vorwärts und rückwärts durchlaufen. Der Knopf Hilfe zeigt diese Hilfe. Der Knopf Chronik zeigt eine Chronik der bereits betrachteten Seiten, und der Knopf Lesezeichen eine Liste von Lesezeichen, die sie bisher angelegt haben.

URL-Eintragsfeld

Das Eintragsfeld erlaubt die direkte Eingabe von URLs. Der Hilfebrowser akzeptiert folgende normale und `magische' URL-Typen:

http:
Über das HT-Transferprotokol auf eine Seite zugreifen.

Beispiele:
  • http://www.gnome.org
  • http://www.gnome.org/index.shtml
  • http://www.gnome.org/index.shtml#majic
file:
Auf eine Datei im lokalen Dateisystem zugreifen.

Beispiele:
  • file:/home/msf/maji12.html
  • file:/usr/man/man1/ls.1
  • file:/usr/src/gogo/README
whatis:
Eine Stichwortsuche in Dokumentnamen.

Mögliche Formen sind:
  • whatis:ls
  • whatis:xman
toc:
Ein Inhaltsverzeichnis existierender Dokumente zeigen.

Mögliche Formen sind:
  • toc:
  • toc:man
  • toc:info
  • toc:ghelp
man:
Eine Handbuchseite laden.

Beispiele:
  • man:ls
  • man:ls(1)
info:
Eine GNU-Info-Seite laden.

Beispiele:
  • info:emacs
  • info:emacs#Top
ghelp:
Die GNOME-Hilfe für ein Programm anzeigen.
HINWEIS: Es existiert noch nicht viel Dokumentation in diesem Format.

Beispiele:
  • ghelp:help-browser

Lesezeichen-Fenster

Das Lesezeichen-Fenster listet alle momentan verfügbaren Lesezeichen auf. Um ein Lesezeichen hinzuzufügen, benutzen Sie die Option Lesezeichen hinzufügen im Menü Datei.

Chronikfenster

Das Chronikfenster listet eine Chronik aller Seiten auf, die Sie nach und nach besucht haben. Sie hat eine maximale Länge, die Sie unter Einstellungen konfigurieren können.
