Documentación del Visualizador de Ayuda de GNOME

El Visualizador de Ayuda de GNOME es una interface que le permite acceder a la documentación en varios formatos presente en su computadora, y en otras computadoras a través de Internet. Este pequeño documento describe las particularidades principales del Visualizador de Ayuda de GNOME.

Opciones de la Línea de Comando

      -d, --debug                Mensajes de depuración
  -h ALTO                    Alto de la ventana
  -w ANCHO                   Ancho de la ventana
  -x X                       Posición X de la ventana
  -y Y                       Posición Y de la ventana
      --display=DISPLAY      Display X11 a usar
      --no-xshm              No usar la extensión de memoria compartida XSHM
  -?, --help                 Mostrar esta ayuda
      --usage                Mostrar un corto mensaje sobre el uso
  -V, --version              Mostrar versión del programa

Menues

Hay varias entrades de menú situadas en la parte superior de la ventana de aplicación del Visualizador de Ayuda de GNOME. Sus funciones deberían ser evidentes.

Barra de Herramientas

The toolbar has several buttons to make using the Help Browser much more convenient. The Next and Prev button cause the browser to go back and forth through the history of the pages you have visited during this session. The Help button brings up this help. The History> button brings up a history of the sites you have visited, and the BMarks button brings up a bookmark listing of sites you've marked in the past.

URL Entry Box

The entry box allows you to enter URL's directly. The Help Browser recognizes the following standard and 'magic' URL types:

http:
Para acceder documentos usando el protocolo HTTP.

Por ejemplo:
  • http://www.gnome.org/
  • http://www.gnome.org/index.shtml
  • http://www.gnome.org/index.shtml#majic
file:
Para acceder a archivos del sistema de archivos local.

Por ejemplo:
  • file:/home/msf/maji12.html
  • file:/usr/man/man1/ls.1
  • file:/usr/src/gogo/README
whatis:
Do a subtring search of document names.

Possible forms are:
  • whatis:ls
  • whatis:xman
toc:
Load a table of contents of available documents.

Possible forms are:
  • toc:
  • toc:man
  • toc:info
  • toc:ghelp
man:
Load a manual page.

Examples are:
  • man:ls
  • man:ls(1)
info:
Load a GNU info page.

Examples are:
  • info:emacs
  • info:emacs#Top
ghelp:
Load a GNOME help for an application.
NOTE:Not much documentation in this format exists currently.

Examples are:
  • ghelp:help-browser

Marca Páginas

The bookmark window lists all the bookmarks currently available. To add a bookmark use the Add Bookmark option under the File menu.

Historia

The history window lists a history of the sites which have been visited over time. It has a maximum length configurable under the Configure option of the File menu.
